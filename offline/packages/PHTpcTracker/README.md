# PHTpcTracker

PHTpcTracker is a TPC track reco package written for sPHENIX collaboration

## Features
   - CA-based Seed Finder
   - GenFit-based Track Follower
   - Rave-based Vertex Finder
   - JSON-based Event Exporter (event => Event Display format)

## Install

Pre-requisite: g++11

How-to Compile:
```sh
$> make
```
